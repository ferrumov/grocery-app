# Grocery List App

## Installation

#### 1. Install dependencies:

```bash
  npm install
```

#### 2. Run JSON Server:

```bash
  npm run server:dev || npm run server:prod
```

#### 2. Run Expo Go (in another terminal window):

iOS:

```bash
  npm run ios:dev || npm run ios:prod
```

Android:

```bash
  npm run android:dev || npm run android:prod
```
